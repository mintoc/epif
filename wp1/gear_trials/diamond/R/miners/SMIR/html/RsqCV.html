RsqCV {SMIR}R Documentation

Cross-validated coefficient of determination

Description

This function provides the leave-one-out crossvalidation version of the coefficient of determination for regression models

Usage

RsqCV(model)

Arguments

model and object as returned by 'lm'

References

Aitkin, M., Francis, B., Hinde, J. and Darnell, R. (2008). Statistical modelling in R, UOP.

Examples

data(trees,package="SMIR")
RsqCV(lm(v~h+d,data=trees))

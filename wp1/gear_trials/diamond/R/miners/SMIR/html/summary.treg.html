summary.treg {SMIR}R Documentation

Summarizing t-regression model fits

Description

The functions are all methods for class "treg" model objects.

Usage

summary.treg(object)

Arguments

object An object of class "treg"

Details

Displays the results from a fit of the $t$-distribution for known degrees of freedom.

Author(s)

ross.darnell@csiro.au

References

Aitkin, M., Francis, B., Hinde, J. and Darnell, R. (2008). Statistical modelling in R, OUP.

See Also

SMIR::treg

Examples

library(SMIR)
data(stackloss)
stackloss.lm <- lm(y ~ x1 + x2 + x3, data = stackloss)
(stackloss.treg1.1 <- treg(stackloss.lm , r=1.1, verbose = FALSE) )
summary(stackloss.treg1.1)
